Lecture Slides: Remote Sensing Instrumentation

This slide deck introduces to some common sensors used in remote sensing analyses, including Landsat, Sentinel-2, MODIS, and PlanetScope, with an emphasis on comparing and contrasting the use cases of each satellite data product. The lecture ends with students using what they've learned so far to decide which sensor they would use for a variety of remote sensing case studies.

In-Class Discussion Activity: Choosing a Sensor

Following the above lecture, students work in small groups to discuss what data products they would use for the provided analyses. This exercise is designed to encourage students to think creatively about what data best fits different kinds of problems, and to prepare them to be problem solvers when working with remotely sensed imagery.
